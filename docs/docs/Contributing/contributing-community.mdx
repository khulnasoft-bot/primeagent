There are several ways you can interact with the Primeagent community and learn more about the Primeagent codebase.

## Join the Primeagent Discord server

Join the [Primeagent Discord Server](https://discord.gg/EqksyE2EX9) to ask questions and showcase your projects.

## Follow Primeagent on X

Follow [@primeagent_ai](https://twitter.com/primeagent_ai) on X to get the latest news about Primeagent.

## Star Primeagent on GitHub

If you like Primeagent, you can star the [Primeagent GitHub repository](https://github.com/khulnasoft/primeagent).
Stars help other users find Primeagent more easily, and quickly understand that other users have found it useful.

Because Primeagent is open-source, the more visible the repository is, the more likely the codebase is to attract [contributors](/contributing-how-to-contribute).

## Watch the GitHub repository

You can watch the [Primeagent GitHub repository](https://github.com/khulnasoft/primeagent) to get notified about new releases and other repository activity.

To get release notifications only, select **Releases only**.

If you select **Watching**, you will receive notifications about new releases as well as issues, discussions, and pull requests, if you are interested in that activity.
For information about customizing repository notifications, see the [GitHub documentation on repository subscriptions](https://docs.github.com/en/account-and-profile/managing-subscriptions-and-notifications-on-github/managing-subscriptions-for-activity-on-github/viewing-your-subscriptions).

## Request enhancements and get help through GitHub

You can also submit feature requests and get help with Primeagent through the GitHub repository.
For more information, see [Get help and request enhancements](/contributing-github-issues).
